      // ...
    },
    {
      "id": "103188971072973252",

## 属性

### `ancestors` {#ancestors}

**描述:** 嘟文串中的上级嘟文。\
**类型:** [Status]({{< relref "entities/Status" >}})数组\
**版本历史:**\
0.6.0 - 添加

### `descendants` {#descendants}

**描述:** 嘟文串中的下级嘟文。\
**类型:** [Status]({{< relref "entities/Status" >}})数组\
**版本历史:**\
0.6.0 - 添加
